# Code of conduct

Be decent. This project is worked on by a small number of people in their spare time; treat everyone here the way you would want to be treated.

- Focus on the work. Critique code and ideas, not people.
- No harassment, personal attacks, or discrimination of any kind.
- Assume good faith and keep disagreements technical.

Behaviour that makes the project hostile to work in is grounds for having contributions declined and, if it continues, being blocked from the repository.

Report problems privately to **bakemonochan@proton.me**. Reports are handled discreetly
